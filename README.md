# OpenGL Settings Manager PRO
Профессиональный инструмент для мониторинга и настройки OpenGL/Mesa в Linux.

## Возможности
- Живые графики нагрузки CPU и RAM
- Управление параметрами Mesa (VSync, HUD)
- Мониторинг температуры и процессов GPU
- Экспорт системных отчетов

## Установка
1. sudo apt install python3-tk mesa-utils lsof lm-sensors
2. pip install -r requirements.txt
3. python3 main.py
